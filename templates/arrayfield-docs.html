  • üé® Fully Themeable ‚Äî Multiple built-in themes with custom theme support, switchable in-app

  • üîì Open Source ‚Äî Fully open source and free to use

Getting Started

To get started with ArrayField, check out the sidebar on the left. It provides quick links to help you use all ArrayField features in your project. Keep this documentation handy so you can refer to it as you build.

Booting the Library

Loading ArrayField

local ArrayField = loadstring(game:HttpGet('https://raw.githubusercontent.com/vqmpjayZ/laboratory/refs/heads/main/ArrayField/ArrayField.lua'))()

Enabling Configuration Saving

üí° tip
  1. Set a FolderName and FileName in the ConfigurationSaving table of CreateWindow
  2. Choose a unique Flag identifier for each supported element you create
  3. The user can enable or disable config saving from the in-app settings panel

ArrayField will automatically save and load the configuration when the user enables it from settings.

‚Ä∫Configuration ‚Ä∫Windows

Windows

Creating a Window

local Window = ArrayField:CreateWindow({
   Name = "My Script Hub",
   Icon = "shield", -- Lucide icon name or Roblox asset ID
   LoadingTitle = "Loading Script Hub",
   LoadingSubtitle = "by YourName",

   ConfigurationSaving = {
      FolderName = "MyScriptHub",
      FileName = "Config"
   }
})

Creating a Tab

local Tab = Window:CreateTab("Tab Example", "home") -- Title, Image

Lucide Icon Support

You can use Lucide Icons by passing the icon name as a string.

local Tab = Window:CreateTab("Combat", "sword")

You can also use Roblox asset IDs:

local Tab = Window:CreateTab("Tab Example", 4483362458)

All Lucide Icons ¬∑ Supported Lucide Icons

Creating a Category

Categories organize tabs into collapsible groups in the sidebar.

local MainCategory = Window:CreateCategory("Main Features", {DefaultOpen = true})

Pass the category as the third argument to CreateTab:

local PlayerTab = Window:CreateTab("Player", "user", MainCategory)

Creating a Section

local Section = Tab:CreateSection("Section Example")

Updating a Section

Section:Set("New Title")

Separators & Spacing

Tab:CreateSeparator(Section, {
    Title = "Advanced Settings",
    Width = 0.9,
    Padding = 10
})

Tab:CreateSpacing(Section, 15) -- 15 pixels of vertical space

Visibility & Destroying

ArrayField:SetVisibility(false)

ArrayField:IsVisible()

ArrayField:Destroy()

Themes

All themes are designed to display ArrayField in its best form while allowing personality within each script. Users can switch themes at any time from the in-app settings panel.

Using Themes

You can optionally set a default theme via DefaultTheme in CreateWindow:

local Window = ArrayField:CreateWindow({
    Name = "My Hub",
    DefaultTheme = "Ocean"
})
‚ö†Ô∏è note

Setting DefaultTheme overrides the user's previously selected theme on every execution. Omit it to let users keep their choice.

Themes Available

Console
‚Ä∫UI Components‚Ä∫Console

Console

A built-in terminal-style log viewer with color coded entries, timestamps, auto-scroll, and a configurable max line limit.

Creating a Console

local Console = Tab:CreateConsole({
    Title = "Remote Logger",
    Icon = "terminal",
    MaxLines = 200,
    Height = 150,
    Timestamps = true,
    SectionParent = Section
})

Logging Methods

Console:Log("Script initialized")       -- Gray   [LOG]
Console:Info("Listening for remotes")  -- Blue   [INFO]
Console:Success("Connected")         -- Green  [OK]
Console:Warn("Rate limit near")      -- Yellow [WARN]
Console:Error("Failed to fetch")     -- Red    [ERROR]

Console:Clear()
üí° tip

The console has a small trash icon in the top right that users can click to clear all entries. It also auto-scrolls to the latest entry and removes old entries when MaxLines is exceeded.

Saving & Loading
‚Ä∫Reference‚Ä∫Saving & Loading

Saving & Loading

When ConfigurationSaving is set and the user enables it from settings, ArrayField automatically saves and loads element values using Flag identifiers.

ArrayField.SaveConfiguration()
ArrayField.LoadConfiguration()

print(ArrayField.Flags["SpeedToggle"].CurrentValue)

Destroying the UI

ArrayField:Destroy()

Completely removes the interface and cleans up all connections.
