Booting the Library

Loading ArrayField

Enabling Configuration Saving

üí° Tip

Configuration saving is toggled by the user from the in-app settings panel. As a developer, you just need to set a FolderName and FileName in ConfigurationSaving, and assign a unique Flag to each element you want saved.

Windows

Creating a Window

local Window = ArrayField:CreateWindow({
    Name = "My Script Hub",
    Icon = "shield",
    LoadingTitle = "Loading Script Hub",
    LoadingSubtitle = "by YourName",
    ConfigurationSaving = {
        FolderName = "MyScriptHub",
        FileName = "Config"
    }
})
Parameter Type Description
Name string Window title displayed in the topbar
Icon string | number Lucide icon name or Roblox asset ID for the topbar icon
LoadingTitle string Text shown during the loading screen
LoadingSubtitle string Subtitle shown during the loading screen
DefaultTheme string Optional. Forces a theme and prevents users from keeping their previously selected one
ConfigurationSaving table Contains FolderName and FileName for config file storage

ArrayField:SetVisibility(false)

local isVisible = ArrayField:IsVisible()

Tabs

Creating a Tab

local Tab = Window:CreateTab("Main", "home")

The first argument is the tab name, the second is an optional icon (Lucide icon name or Roblox asset ID).

Lucide Icon Support

ArrayField supports Lucide Icons. Pass the icon name as a string:

local CombatTab = Window:CreateTab("Combat", "sword")
local VisualsTab = Window:CreateTab("Visuals", "eye")
local PlayerTab = Window:CreateTab("Player", "user")

You can also use Roblox asset IDs:

local Tab = Window:CreateTab("Tab Example", 4483362458)

Browse all Lucide Icons ¬∑ Supported Icons List

üí° Marquee Effect

If a tab name is too long to fit in the sidebar, it automatically scrolls horizontally so the full name is always visible.

Categories

Categories let you organize tabs into collapsible groups in the sidebar.

Creating a Category

local MainCategory = Window:CreateCategory("Main Features", {DefaultOpen = true})
local SettingsCategory = Window:CreateCategory("Settings", {DefaultOpen = false})

Creating Tabs Under Categories

Pass the category as the third argument to CreateTab:

local PlayerTab = Window:CreateTab("Player", "user", MainCategory)
local ConfigTab = Window:CreateTab("Config", "settings", SettingsCategory)
Parameter Type Description
Name string Category display name
DefaultOpen boolean Whether the category starts expanded or collapsed

Sections

Sections organize elements within a tab into titled groups.

Creating a Section

local Section = Tab:CreateSection("Movement")

Updating a Section

Section:Set("New Section Title")

To place elements inside a section, pass SectionParent = Section in the element's settings table.

Themes

ArrayField ships with multiple built-in themes. Users can switch themes at any time from the in-app settings panel without needing to re-execute.

Setting a Default Theme

You can optionally force a default theme in CreateWindow:

local Window = ArrayField:CreateWindow({
    Name = "My Hub",
    DefaultTheme = "Ocean",
})
‚ö†Ô∏è Note

Setting DefaultTheme overrides the user's previously selected theme on every execution. Omit this parameter to let users keep their choice.

Universal Methods

All interactive elements support these common methods:

Element:Lock("Reason text")

Element:Unlock()

Element:Visible(false)
Element:Visible(true)

Element:Destroy()

Saving & Loading

When ConfigurationSaving is set and the user enables it from settings, ArrayField automatically saves and loads element values using the Flag identifiers.

ArrayField.SaveConfiguration()

ArrayField.LoadConfiguration()

print(ArrayField.Flags["SpeedToggle"].CurrentValue)
print(ArrayField.Flags["SpeedValue"].CurrentValue)

Destroying the UI

ArrayField:Destroy()

Completely removes the interface and cleans up all connections.
