API Information

Use the API directly with crop options:

GET /convert-image?url=YOUR_IMAGE_URL&crop=true
GET /convert-image?url=YOUR_IMAGE_URL&crop=false

• crop=true: Smart crop (default) - maintains aspect ratio
• crop=false: Simple resize - stretches to fit exactly
Remember to remove the /converter part while doing this!
